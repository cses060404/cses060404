Mobile Browsing - Analysis

Introduction

In the task, we were asked to use our mobile devices to browse for 24 hours instead of using

desktop or laptop. The purpose of this task is to help us having a better understanding of

how it is like to browser in a tiny screen for a long time. I was able to pay more attention

on the sites that I have always been browsing and I did have different feelings about those sites.

Here are some of the example questions to help us know how the experiment went.

Questions to consider:

  • Can you still see what you want to see?
  • Can you find what you're looking for?
  • Can you do what you want to do?
  • Any frustrations?
  • Anything work better on the small screen?

Findings

Can you still see what you want to see?

I can still pretty much see all the information that I wanna to see. However, for the website that I use the most -- BYUI I-Learn, I cannot see the couse links but only the assignments. Sometimes I just want to view a specific couse and the grade of it, without the tab, it's harder to gain access to it.
